Shodan-Style Network Search Engine

Used for vulnerability analysis and penetration testing

Add the IP subnets that you would like to scan to the ips.txt file and then run scanner.py.

The scanner will then gather information and then add it to a MongoDB database. Use this site to search it!

Here are the search options available:

Search by Title

Search by Domain

Search by Header Response

Search by IP Address

Search by Port

Search by Header Key Response
